# 一个基于vue3+typescript的画板 (学习canvas + vue3 + typescript)

## 项目目录结构
```
└── WhiteBoard                    // 画板组件
    ├── components                // 拆分的组件
    ├── config                    // 参数配置项
    ├── directives                // 指令
    ├── hooks                     // 根据业务功能拆分的逻辑代码
    ├── icons                     // 图标
    ├── types                     // 定义的类型
    ├── utils                     // 通用工具方法
    └── index.vue                 // 插件主入口文件
```
